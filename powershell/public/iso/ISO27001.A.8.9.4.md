Allowing any user to register applications enables low-privilege users to create service principals with arbitrary permissions. Malicious insiders or compromised users can register apps as a persistence mechanism or to request elevated permissions via admin consent.

#### Remediation action:

Configure via: Entra admin center > Users > User settings
