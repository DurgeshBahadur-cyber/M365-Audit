# M365Advisor - PowerShell Commands

## Overview
This page lists all the commands available in the M365Advisor PowerShell module.

The M365Advisor PowerShell module is a collection of commands that help you write Pester tests to validate your Microsoft 365 tenant configuration. You can view the Graph API calls for these commands in the [M365Advisor GitHub repository's src folder](https://github.com/m365advisor365/m365advisor/tree/main/src/public). You can also select the `Edit this page` link found in the command's documentation to view the source code for the command.

See the [writing M365Advisor tests](/docs/writing-tests) guide for more information on how to write your own tests using these commands.
