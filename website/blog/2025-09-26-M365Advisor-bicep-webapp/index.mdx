## New version of M365Advisor on Azure Web App 🎉

![M365Advisor Bicep Web App](./img/m365advisor-webapp-header.png)

### What was it before?
The setup allowed M365Advisor to run on an Azure Web App, enabling you to perform Microsoft Graph security configuration checks on your Microsoft tenant directly from the Azure Web App, secured with Entra ID Authentication through an Azure Bicep deployment. At that time, it only supported Microsoft Graph security configuration tests. Many people asked me to update the deployment to include also the other tests that are part of M365Advisor, because we want to utilize all the available tests that M365Advisor has.

### What has changed?
- 🔥 Support for the latest M365Advisor PowerShell module through the use of PowerShell 7.4 runtimes. 
- 💪🏻 Support for Exchange Online security tests 
- 🎉 Support for Security & Compliance security tests 
- 📱Support for Microsoft Teams security tests 
- 🤙🏻 Support for Azure configuration security tests

### 📚 **Documentation**
