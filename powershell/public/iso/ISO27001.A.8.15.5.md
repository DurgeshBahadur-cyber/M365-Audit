Audit log search is the investigative capability for Microsoft 365 — every admin action, user sign-in, file access, and configuration change is recorded here. Without it enabled, incidents cannot be investigated.

#### Remediation action:

Configure via: Microsoft Purview > Audit > Start recording user and admin activity
