SharePoint is a primary repository for sensitive organizational documents. Unrestricted external sharing allows any user to share any document with any email address, bypassing data classification controls and creating permanent unauthenticated access links.

#### Remediation action:

Configure via: SharePoint admin center > Policies > Sharing
