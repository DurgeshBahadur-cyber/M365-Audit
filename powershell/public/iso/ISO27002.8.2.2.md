Applications with dangerous application permissions (not delegated) act as autonomous service accounts that can read, write, and delete data for all users without any user sign-in. Foreign (multi-tenant) apps with these permissions operate from outside the tenant's control plane.

#### Remediation action:

These permissions have documented attack paths to Global Administrator. Remove or replace with least-privilege alternatives

Configure via: Entra admin center > Enterprise applications > review foreign apps with Tier 0 permissions
