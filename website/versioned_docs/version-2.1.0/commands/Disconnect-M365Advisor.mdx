## SYNOPSIS

Helper method to sign out of the current Microsoft Graph session.
Alternate for Disconnect-MgGraph.

## SYNTAX

```powershell
Disconnect-M365Advisor [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Use this cmdlet to sign out of the current Microsoft Graph session.

This cmdlet is a helper method for running the following command.
```
Disconnect-MgGraph
```

## EXAMPLES

### EXAMPLE 1

```powershell
Disconnect-MtGraph
```

### EXAMPLE 2

```powershell
Disconnect-M365Advisor
```

### EXAMPLE 3

```powershell
Disconnect-MtM365Advisor
```
