The command `Send-MtTeamsMessage` to post your M365Advisor results to a Teams channel has been there for a while but we did forget to write the documentation for it. This is now fixed!

![M365Advisor - Microsoft Teams Alerts](./img/m365advisor-teams-alert.png)

{/* truncate */}

You can find the steps in the [Teams Alerts](/docs/next/alerts/teams) page.

A huge shout out to [Guido Baijense](https://blog.pentiago365.nl/) for contributing the code for this feature!

This is what I really ❤️ about the M365Advisor community. 🙏

Next up, we need to add this as configurable option to the GitHub Action. This way you can easily enable it for your workflows without having to write scripts. If you are interested in helping out with this, check out the [action.yml](https://github.com/m365advisor365/m365advisor/blob/main/action.yml) that needs to be updated.

Reach out on the M365Advisor channel on our [Discord server](https://discord.com/channels/1125617152368594976/1226351860693205062) if you have any questions or feedback.
