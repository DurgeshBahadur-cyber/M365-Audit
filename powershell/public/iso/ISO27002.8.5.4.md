LAPS randomizes the local administrator password on each managed device, preventing credential reuse attacks. Without LAPS, all managed devices share the same local admin password — if one device is compromised, that credential provides local admin access to every other device.

#### Remediation action:

Configure via: Entra admin center > Devices > Device settings > Enable Microsoft Entra Local Administrator Password Solution (LAPS) > Yes
