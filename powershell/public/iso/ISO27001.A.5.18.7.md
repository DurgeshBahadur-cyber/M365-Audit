Global Administrators control every aspect of the tenant. Standard MFA can be bypassed via adversary-in-the-middle (AiTM) phishing attacks that relay the authentication session. Phishing-resistant MFA (FIDO2, Windows Hello, certificate) eliminates this attack class for the highest-privilege accounts.

#### Remediation action:

Enroll all Global Administrators in phishing-resistant MFA (FIDO2, Windows Hello for Business, or certificate-based)

Configure via: Entra admin center > Protection > Authentication methods > Policies
