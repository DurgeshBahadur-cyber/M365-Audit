Per-user MFA is a legacy enforcement method that applies MFA independently of Conditional Access. It does not support modern features like risk-based enforcement, device trust, or named location exceptions. Running both per-user MFA and CA simultaneously creates conflicts and gaps.

#### Remediation action:

Use Conditional Access policies for MFA enforcement instead of per-user MFA

Configure via: Entra admin center > Users > Per-user MFA > Ensure all users show Disabled
