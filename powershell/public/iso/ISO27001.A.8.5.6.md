Apps with active client credentials (secrets or certificates) have a persistent authentication mechanism that does not require user interaction. A leaked or brute-forced credential provides persistent API access without any user sign-in event or MFA challenge.

#### Remediation action:

Remove secrets/certificates from apps that no longer need them

Configure via: Entra admin center > Enterprise applications > review each app with credentials
