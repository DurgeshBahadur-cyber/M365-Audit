SharePoint file storage can be used to host and distribute malware if infected file quarantine is disabled. A malware-infected document uploaded to SharePoint can be synced to all users who have access to the library, spreading the infection across the organization.

#### Remediation action:

Configure via: SharePoint admin center > Policies > Malware protection
