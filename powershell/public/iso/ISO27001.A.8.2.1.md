Applications with Tier 0 permissions (e.g., Directory.ReadWrite.All, RoleManagement.ReadWrite.Directory) can read and modify the entire Entra ID directory, including assigning admin roles to other principals. A compromised app credential is equivalent to a compromised Global Administrator.

#### Remediation action:

Each has a documented path to Global Administrator. Replace with narrower permissions or use managed identities where possible

Configure via: Entra admin center > App registrations > review internal apps with Tier 0 permissions
