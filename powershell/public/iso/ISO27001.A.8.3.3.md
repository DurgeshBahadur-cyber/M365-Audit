Shareable links that grant access to 'anyone with the link' bypass all identity and device controls. A single leaked link is sufficient to expose the target content to any person, device, or automated system worldwide.

#### Remediation action:

Configure via: SharePoint admin center > Policies > Sharing > File and folder links > Default link type > Specific people
